DNS server because your computer doesn't naturally understand human language. This DNS server takes "google.com" and turns it into an address, called an IP address. This address acts like home addresses work for us, it tells the computer where it needs to go next. So this computer sends a request to this address, and the servers at this address present your computer with something called and HTML file, and its accompanying files. The browser then has to take this file, and turns it into something that looks nice to humans like us. Some websites have extra capability, like a live chat, or telling you the weather. This is usually done using a language called JavaScript, this language can be used to make small programs that your computer executes to make the website more interactive.
