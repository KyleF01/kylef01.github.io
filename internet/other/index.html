How to make a Website

So on the home page I made a fairly beefy promise. To teach how to make a website of your very own. This will be a very simplistic webpage, but it will be one of your own.

First, create a new file and name it "index.html", without the quotation marks of course. Then you add some tags to tell your browser some info about your page. Start with <!DOCTYPE html> this tells your browser to treat this as an HTML file.

Next, make <html> and </html> tags, you will put everthing inside of these tags, so your browser knows what to show.

Now you want to put <head> and </head>, this is where you will put all of the more technical stuff, like a title. Underneath this you will put <body> and </body> tags. Inside these tags are where you will create your page.

There are plenty more tags I could talk about, but it would take years. Instead I will leave you with an example, and show some of the more common tags.


    List of Common Tags
  • p, Paragraph tag, used for text
  • a, Link, used with href tag to link to another page
  • h1, Header, makes text bigger
  • ul and li, Unordered list and Iist Items, Makes a bulleted list like this one
  • title, Sets the title of the page at the top of the tab.
