HTML

HTML is a vital part of the internet, infact it is what I wrote this website in. HTML stands for Hyper Text Markup Language, and uses things called elements. Each element is surrounded by "tags" such as <b>Hi!</b>. For example, those "b" tags would turn the text in there, 'Hi!', bold. There are other tags, such as the script tag, in which you specify JavaScript code to run on your website, to increase interactivity.

HTML is sent exactly the way it is written, tags and all. It is up to your browser to take each element and do exactly what the tags say with them. For example: if the buttons above worked, or you see a picture on the button on the far left of this website, then the tags worked!
